# Project 3 – AWS VPC Networking Lab

## Objective

Build a custom VPC with public and private subnets to understand AWS networking fundamentals.

## Services Used

* Amazon VPC
* Subnets
* Internet Gateway
* Route Tables
* Security Groups
* EC2 (later)

## Architecture

Internet
|
Internet Gateway
|
VPC (10.0.0.0/16)
|
├── Public Subnet (10.0.1.0/24)
|
└── Private Subnet (10.0.2.0/24)

## Key Learnings

* CIDR planning
* VPC creation
* Public vs Private Subnets
* AWS network isolation
