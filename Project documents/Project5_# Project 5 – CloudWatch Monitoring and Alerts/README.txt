# Project 5 – CloudWatch Monitoring and Alerts

## Objective

Monitor EC2 performance and receive automated alerts when CPU usage exceeds a threshold.

## Services Used

* Amazon CloudWatch
* Amazon SNS
* Amazon EC2

## Monitoring Workflow

EC2 Instance
|
CloudWatch Metrics
|
CloudWatch Alarm
|
SNS Topic
|
Email Notification

## Key Learnings

* CloudWatch Metrics
* Alarm Creation
* SNS Notifications
* CPU Monitoring
* Basic AWS Operations Monitoring

## Outcome

Successfully monitored an EC2 instance and received automated email alerts when CPU utilization exceeded the configured threshold.
